# Dots Generator

Générateur de backgrounds à base de dots, exportables en **PNG** ou **SVG**. Un seul fichier HTML, sans dépendance.

## Fonctionnalités

- **10 dispositions** : Grille, Disque, Anneau, Spirale, Sphère 3D, Perspective, Ondes, Gouttes, Vortex, Topographie
- Formes de dots : carré, arrondi, cercle
- Contrôle de la taille, densité, clustering, fondu, palette de couleurs, opacité, fond
- **Seed** reproductible + bouton « Tout randomiser »
- Aperçu live et export PNG (1×–4×) / SVG
- Dimensions personnalisables avec presets

## Utilisation

Ouvrir `index.html` dans un navigateur, ou visiter la version en ligne (GitHub Pages).

## Développement

Aucune build nécessaire — tout tient dans `index.html`. Pour servir localement :

```bash
npx serve .
```
